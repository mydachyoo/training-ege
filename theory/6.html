Для решения этого задания необходимо уметь пользоваться программным обеспечением, в частности средой разработки языка программирования(C++, Python, Pascal).

Рекомендуется Python или Pascal, из-за простоты их использования.

Для решения этого задания, необязательно иметь навык программирования.

Первая задача:

Ниже приведена программа, записанная на нескольких языках программирования:

Pascal Python C++

begin
readln(s);
readln(t);
if (s > 10) or (t > 10)

 else
writeln("NO")
end.

t = int(input())

 print("YES")

#include 
using namespace std;
int main(){
 int s, t;
 cin >> s;
 cin >> t;
 if (s > 10 || t > 10)
   cout << "YES" << endl;
 else
   cout << "NO" << endl;
 return 0;
 }

Было проведено 9 запусков программы, при которых в качестве значений переменных вводились следующие пары чисел (s, t):

(1, 2); (11, 2); (1, 12); (11, 12); (–11, –12); (–11, 12); (–12, 11); (10, 10); (10, 5)

Сколько было запусков, при которых программа напечатала «YES»?

Решение:

  • Программа напечатает «YES», если одна из введённых переменных s или t будет больше 10.
  • Значит, было 5 запусков, при которых программа напечатала «YES». В качестве значений переменных s и t в этих случаях вводились следующие пары чисел:
    (11, 2); (1, 12); (11, 12); (–11, 12); (–12, 11)

Ответ: 9

Альтернативное решение:

  • Решение, заключается в переписывание кода Python в Интерпретатор, на экзамене это средство разработки Python IDLE, и последующего запуска программы.

Ответ: 9
